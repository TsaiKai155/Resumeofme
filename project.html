ASP.NET MVC專案簡介

這個專案展示了我在資料分析與串接資料庫與後台網站設計的成果。 左邊影片提供完整的介紹與操作示範，右邊則是簡要說明專案背景與應用情境。

使用技術

-前端:HTML / CSS / JavaScript / Bootstrap
-後端:ASP.NET Core MVC / C# / SQL Server


- 成果：前後端整合後，系統能提供穩定、安全且舒適的使用者體驗，滿足天文館日常營運與資訊服務的需求，系統蒐集票務、商城銷售與活動參與等關鍵數據，並生成視覺化報表與分析圖表。業者可依據這些洞察， 精準規劃未來活動、最佳化資源配置、制定有效的行銷策略，大幅降低經營風險並提升決策品質。
